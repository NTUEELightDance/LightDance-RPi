# 2021-LightDance-RPi

### Setup zeromq

You must first install `cppzmq` to enable the usage of controller

-   Installation on Mac:
    Type in the following script:

    ```shell
    cd LightDance-RPi/scripts/
    bash install_zeromq_mac.sh
    ```

-   Installation on RPi:
    Type in the following script:

    ```shell
    cd LightDance-RPi/scripts/
    bash install_zeromq_rpi.sh
    ```
